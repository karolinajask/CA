User requirements for the Cars Marketplace Information System

Registered users’ information management: The system should be able to store and manage registered users’ records, including personal details, search history, viewed ads and the ads added by them.
Ads management: The system should allow registered users to add/delete ads.
Communication management: The system should allow communication between registered users which could be email notifications, text messages or live chat.
Access control management: The system should permit registered users only to add new ads ,contact a user who posted an ad or report a fraudulent ad. All users should be able to view, sort and filter ads.
Transaction management: The system should be able to allow registered users to transact with each other and perform basic verification of transaction details.
Security management: The system should include different permissions for unregistered users, registered users, and system administrator.
